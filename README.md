# AI온보딩\_대전4반\_서보민\_박정수\_전홍선

## LocalHub

대전·충청 지역의 맛집, 관광지, 지역 정보를 공유할 수 있는 커뮤니티 서비스입니다.

사용자는 지도에서 주변 맛집과 관광지를 확인하고, 장소별 리뷰를 작성할 수 있습니다. 또한 게시판을 통해 지역 정보를 공유하고 AI 챗봇을 이용할 수 있습니다.

## 주요 기능

- 대전·충청 지역 맛집 및 관광지 지도 조회
- 장소 검색 및 카테고리 필터
- 맛집·관광지 랭킹 조회
- 장소별 리뷰 작성, 수정, 삭제
- 지역 정보 게시판
- 게시글 및 댓글 작성, 수정, 삭제
- AI 챗봇 대화 기능

## 기술 스택

### Frontend

- Vue 3
- Vite
- Vue Router
- Kakao Maps API
- Axios / Fetch API

### Backend

- FastAPI
- SQLAlchemy
- SQLite3
- OpenAI API

## 팀원

- 서보민
- 박정수
