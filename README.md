# LocalHub

> 대전·충청 지도 기반 지역 정보 공유 서비스

LocalHub는 대전·충청 지역의 맛집, 관광지, 방문 후기, 지역 커뮤니티 정보를 하나의 서비스에서 확인하고 공유할 수 있도록 만든 웹 서비스입니다.

## 배포 주소

- https://local-hub-7gky.vercel.app/

## 프로젝트 기획 배경

지역과 관련된 장소 정보, 방문 후기, 주민 추천 정보는 많지만 여러 서비스와 커뮤니티에 흩어져 있어 한 번에 확인하기 어렵습니다.

LocalHub는 다음 기능을 하나의 서비스로 통합하여 지역 정보 탐색부터 공유까지 원스톱으로 제공합니다.

- 장소 검색
- 사용자 리뷰
- 지역 커뮤니티
- AI 챗봇

## 핵심 사용자

### 1. 장소 탐색 사용자

대전·충청 지역의 맛집과 관광지를 찾는 사용자입니다.

- 지도 기반으로 주변 장소 검색
- 맛집·관광지 카테고리별 탐색
- 키워드 검색으로 원하는 장소 확인

### 2. 리뷰 확인 사용자

실제 방문 후기를 확인하고 싶은 사용자입니다.

- 별점과 텍스트 후기 확인
- 평점 기반 인기 장소 랭킹 조회
- 리뷰 작성, 수정, 삭제

### 3. 정보 공유 사용자

지역의 맛집과 관광지를 추천하고 다른 사용자와 소통하려는 사용자입니다.

- 지역 정보 게시글 작성
- 게시글과 장소 연결
- 게시글에서 해당 장소의 지도 화면으로 이동
- 댓글과 대댓글을 통한 소통

## 주요 기능

### 지도 기반 장소 검색

- 대전·충청 지역 맛집과 관광지 조회
- 전체, 맛집, 관광지 필터
- 지역명, 역 이름, 장소명 검색
- 지도 이동 범위에 따른 장소 목록 표시
- 장소 선택 시 상세 정보 확인

### 장소별 리뷰

- 장소별 방문 후기 조회
- 별점 및 텍스트 리뷰 작성
- 비밀번호 기반 리뷰 수정 및 삭제
- 리뷰 평점 기반 맛집·관광지 랭킹 조회

### 지역 정보 게시판

- 최근 게시글 조회
- 게시글 작성, 상세 조회, 수정, 삭제
- 게시글 조회수 표시
- 게시글과 장소 연결
- 게시글에서 연결된 장소의 지도 화면으로 이동

### 댓글 및 대댓글

- 게시글별 댓글 작성
- 댓글 수정 및 삭제
- 일반 댓글에 대댓글 작성
- 비밀번호 기반 수정 및 삭제

### AI 챗봇

- 서비스 이용 중 간단한 질문 입력
- 이전 대화 흐름을 유지한 연속 대화
- 지역 정보 탐색을 돕는 챗봇 기능
